Epic

A local candle maker wants to open an online store selling their handmade candles
and other artifacts, they need a database system to use as part of this online shop.
Details of customers need to be stored, products need to be stored and updated,
a record of customer orders also needs to be maintained.
Design the database system for the Candle Shop.

---------------------------------------------------------------------------------------------------

User Stories

    - As a store manager, I want to have a set inventory which are the only items a customer can
      order.

    - As a store manager, I want to be able to create new items to add to the inventory.

    - As a customer, I want to be able to add items from the store to a cart before actually ordering.

    - As a customer, I want to be able to choose shipping method when making an order.

    - As a store manager, I want to be able to see how many of certain item there is in stock.
